Ban

Command

/ban <User*> <Reason> <Delete>

Description

Ban a member from the server

Parameter Details

User

(Mention or ID) - The user to ban (required*)

Reason

(Text) - The reason to ban the user for (optional)

Delete

(Text) - The number of days of the user's messages to delete (optional)

Check

/ban id reason delete

View a global ban for a user or lookup a specific ban
