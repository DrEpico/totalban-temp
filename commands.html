Ban

Command

/ban <User*> <Reason> <Delete>

Description

Ban a member from the server

Parameter Details

User

(Mention or ID) - The user to ban (required*)

Reason

(Text) - The reason to ban the user for (optional)

Delete

(Text) - The number of days of the user's messages to delete (optional)

Perms

Ban Members

Check

Command

/check <ID> <User>

Description

View a global ban for a user or lookup a specific ban

Parameter Details

ID

(Text) - The ban ID (optional. required if user is missing)

User

(Mention or ID) - The user to check for (optional. required if id is missing)
