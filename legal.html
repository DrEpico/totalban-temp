Privacy Policy

For the duration of the time the bot is in any Guild, we (the bot staff team) reserve the right to generate an invite URL, or use an existing one, for said Guild, for investigation purposes, if the Guild is suspected to be in breach of the Discord Terms of Service and/or Community Guidelines.

Upon the end of the investigation, any invite URLs will be discarded, and no longer be made accessible to us (the bot staff team).

Discord Data Tracked

Message Content and Message Attachments

The bot will read this data at any time a Message is a) Sent to Discord; b) Edited by the sending user; The bot does this in order to process AutoModeration requests used by the Guild the Message was sent to.

The Guild may change the settings to this feature, for their specific Guild at any time they see fit.

Basic User Data

The bot will read basic User Data during certain commands that require a user as an Input Method, for example kick or ban, so that the bot can take the appropriate action on the user.

When the report command is used, the bot will send a request to the Bot Staff Team (no non-staff members can access this). This request will contain the ID and Username of both the Reported User and Reporter.

When running the whois command, the bot will display more information about the user (who's ID is provided as a command argument). This information may include the user's avatar, if any is set, the Public Flags associated with their account, for example, HypeSquad Badges and Nitro status, and the time of the user's account creation.

Discord Data Stored

Basic User Data

The bot will store the IDs of the Globally Banned user(s), and the IDs of the Bot Staff Member who added said user to the list (for tracking purposes). The bot will also store a URL to the image(s) used as a proof method for Globally Banning this User. A User may also have numerous infractions added to their ban. Each infraction will require a separate appeal.

Unlike the previous version of this bot, we (The Bot Staff Team) will be very strict when adding a user to the Global Banlist, as to prevent as much abuse as possible.

As the service provided is a Global Banlist bot, a Globally Banned User will require to Appeal for breaking the Discord Terms of Service and/or Community Guidelines, in order for their Discord Data to be removed from our Systems. As previously stated, we will be very strict about this, and the user must be sincerely regretful of their actions to have their data removed.
